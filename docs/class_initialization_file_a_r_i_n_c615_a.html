FileOperationResult InitializationFileARINC615A::getFileSize ( size_t &  fileSize)
overridevirtual

Get total file size in bytes.

This will not return the size of the entire ARINC615A file, but only the size of the class file.

If you want to get the size of the entire ARINC615A file, you must call the function getFileLength() from the BaseFileARINC615A class.

This method is intended to be used during file deserialization.
